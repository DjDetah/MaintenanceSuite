A robust, single-page React application for managing enterprise IT incidents, parts requests, and regional KPIs. Built with **React**, **TypeScript**, **Tailwind CSS**, and **Supabase**.

## Features
- **Dashboard**: Real-time KPIs, Regional Heatmap, and Recidivism Charts.
- **Incident Management**: High-density data grid with detailed modal views.
- **Parts Workflow**: Integrated lifecycle for tracking hardware requests.
- **User Security**: Role-Based Access Control (RBAC) and Row Level Security (RLS).
- **Data Import**: Advanced Excel normalization for multi-source data ingestion.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Set up your Vercel/Local environment variables:
    - `VITE_SUPABASE_URL`
    - `VITE_SUPABASE_ANON_KEY`

3.  **Run Locally**:
    ```bash
    npm run dev
    ```

4.  **Build**:
    ```bash
    npm run build
    ```
